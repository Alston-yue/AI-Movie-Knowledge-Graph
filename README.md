# 🎬 电影领域知识图谱智能问答系统 (Movie-KGQA-System)

这是我的毕业设计项目，旨在构建一个基于领域知识图谱的智能问答系统。

## 🛠️ 技术架构
- **核心后端**: Python, Flask, LangChain
- **图数据库**: Neo4j (GraphRAG)
- **大模型 API**: DeepSeek-V3
- **前端界面**: Vue.js (追求 DeepSeek 风格的极简 UI)
- **数据来源**: TMDB 5000 静态数据集 + TMDB API 实时更新

## 🌟 核心功能
- [x] 基于 Cypher 语句的图数据库自动查询
- [x] 自然语言处理与大模型意图识别
- [x] 电影数据实时云端同步更新
- [x] 响应式 Web 交互界面
